# 15-Puzzle Solver & Visualizer

一个简单的4\*4数字华容道 （15-Puzzle） 求解工具，包含 Python算法库 与 浏览器端交互演示。  
**纯属**闲暇练手之作，目前仅实现基本可运行版本，后续可能会逐步优化，同时完善诸细节以及不明确之处。
[GitHub Pages 页面](https://ymz-w.github.io/15-puzzle/)

## 文件说明

| 文件 | 说明 |
|------|------|
| `15-puzzle.py` | Python 核心类库，包含 Puzzle 状态表示、BFS/IDA* 求解、HP 算法框架 |
| `index.html` | 独立运行的 Web 演示页面，支持随机打乱、手动布局、HP 求解及动画播放 |

## 详细说明

`15-puzzle.py` 无法直接使用。可根据需要修改 `__main__` 中的测试代码，调用 Explorer 类进行求解。

`index.html` Web 端（可视化演示），直接下载 index.html，用浏览器打开即可使用。此文件由AI生成。

`HP算法`: 分层寻路HierarchicalPathfinding， 泛A\* + 剪枝A\* 结合
    - `泛A\*`: 一种目标宽泛的A\*算法
    - `剪枝A\*`: 简化路径的A\*算法

其他算法: 详见 `15-puzzle.py`， 在`index.html`中仅仅提供 `HP算法` 。

其他不做详细说明。

## 已知局限

HP 算法中的泛 A\* 阶段仍采用在线搜索，尚未实现查表，效率有限。并且可能会有隐藏的bug。
面对较复杂的初始状态时，求解可能耗时较长（数十秒至数分钟）。并且无法确保最优解。

界面与交互细节仍有打磨空间。
